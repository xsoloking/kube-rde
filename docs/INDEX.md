## What is KubeRDE?

KubeRDE is an open-source platform that enables teams to create, manage, and access isolated development workspaces running on Kubernetes.

### Quick Start

```bash
git clone https://github.com/xsoloking/kube-rde.git
cd kube-rde
./scripts/quick-start-kind.sh
```

See [documentation](https://github.com/xsoloking/kube-rde/tree/main/docs) for more details.
